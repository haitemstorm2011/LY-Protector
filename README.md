# نظام LYProtector لمكافحة الفيروسات

يُعد نظام LYProtector تطبيقًا مكتبيًا يعمل على نظام التشغيل Windows، تم تطويره كمشروع تخرج لطلبة قسم علوم الحاسوب. يهدف المشروع إلى تطبيق مفاهيم هندسة البرمجيات وأساسيات أمن المعلومات من خلال تصميم وتنفيذ نظام عملي للكشف عن الملفات الضارة.

---

## هدف المشروع

يهدف هذا المشروع إلى تصميم وبناء نظام مكافحة فيروسات قادر على فحص الملفات واكتشاف البرمجيات الخبيثة باستخدام أسلوب **الكشف المعتمد على التواقيع الرقمية (Signature-Based Detection)**، مع توفير واجهة استخدام سهلة وبنية نظام منظمة تلائم التقييم الأكاديمي.

---

## خصائص النظام

- الفحص السريع والفحص الكامل للنظام
- الكشف عن البرمجيات الخبيثة باستخدام MD5 / SHA
- إدارة قاعدة بيانات تواقيع الفيروسات
- إنشاء تقارير مفصلة لنتائج الفحص
- نظام تفعيل وتوثيق المستخدم
- وحدات الحماية والجدار الناري والخصوصية
- دعم التشغيل بصلاحيات المدير (Administrator)
- واجهة رسومية حديثة ولوحة تحكم متكاملة

---

## التقنيات المستخدمة

- لغة البرمجة: VB.NET  
- الإطار البرمجي: .NET (تطبيقات سطح المكتب)  
- بيئة التطوير: Visual Studio 2022  
- قاعدة البيانات: SQL Server LocalDB  
- خوارزميات التشفير: MD5 / SHA  
- إدارة الإصدارات: GitHub  

---

## تصميم النظام

تم تصميم النظام بالاعتماد على نماذج هندسة البرمجيات القياسية، وتشمل:
- مخططات تدفق البيانات (DFD – المستوى صفر والمستوى الأول)
- مخطط حالات الاستخدام (Use Case Diagram)
- مخطط الكيانات والعلاقات (ERD)
- بنية معيارية متعددة الطبقات
